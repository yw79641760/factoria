## 📋 新架构说明

### v1.0（已废弃）- 模板驱动
```
用户需求 → 选择模板 → 填充变量 → 生成代码
```

### v2.0（当前）- 能力驱动
```
用户需求 → NLU解析 → 识别能力 → 编排能力 → 生成代码
```

---

## 🔑 关键差异

| 维度 | v1.0 模板驱动 | v2.0 能力驱动 |
|------|----------------|-----------------|
| 核心思路 | 选择模板，填充变量 | 识别能力，编排能力 |
| LLM 角色 | 模板填充者 | 能力编排者 |
| 灵活性 | 低（受限于模板） | 高（能力可组合） |
| 扩展性 | 需要新模板 | 沉淀新能力 |
| 可组合性 | 不可组合 | 完全可组合 |
